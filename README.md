## Project Overview

The Birthday App allows users to:

- Enter their name in a form.  
- Receive a random birthday message.  
- See their name displayed with a fun **wave animation**.  
- Enjoy a monochrome or colorful aesthetic with animated balloons.

This project is perfect for beginners to learn **Flask, HTML, CSS animations, and Python logic**.

---

## Features

- Dynamic birthday messages (randomly chosen).  
- Animated wave text for names.  
- Floating balloons animation.  
- Monochrome (black, white, gray) theme available.  
- Responsive and modern design.
