# Ayesha & Ahmed - Wedding Invitation Project

A stunning dual-platform wedding invitation featuring an interactive HTML website and an animated Python Turtle graphics card.

## Files

- `index.html` - Premium HTML wedding invitation website
- `wedding_turtle.py` - Animated Python Turtle wedding card

## HTML Website Features

- **Cinematic Envelope Opening Animation** - Click to open the envelope with smooth flap animation
- **Premium Design** - Glassmorphism effects, gold and rose color scheme, elegant typography
- **Dark/Light Mode Toggle** - Switch between themes with persistent storage
- **Wedding Events Timeline** - Mehndi, Barat, and Walima events with dates and times
- **Live Countdown Timer** - Real-time countdown to the wedding date
- **Confetti Celebration** - Animated confetti on envelope opening
- **Wishes Form** - Guests can send wishes stored locally
- **Sharing Options** - WhatsApp sharing and copy link functionality
- **Responsive Design** - Works perfectly on all devices
- **Floating Particles** - Subtle animated background effects

## Python Turtle Graphics Features

- **Interactive Envelope Opening** - Click anywhere to start the animation
- **Animated Text** - Character-by-character text animation for couple names
- **Decorative Elements** - Hearts and flowers scattered across the card
- **Wedding Events Display** - All three wedding events listed
- **Confetti Animation** - Colorful falling confetti celebration
- **Venue Information** - Wedding location details
- **Blessing Messages** - Traditional wedding blessings

## How to Run

### HTML Website
1. Open `index.html` in any modern web browser
2. Or run a local server:
   ```bash
   python -m http.server 8000
   ```
3. Visit `http://localhost:8000` in your browser

### Python Turtle Card
1. Make sure Python is installed with Turtle graphics
2. Run the script:
   ```bash
   python wedding_turtle.py
   ```
3. Click anywhere on the screen to open the wedding card

## Deployment

### HTML Website
- Simply upload `index.html` to any web hosting service
- Works on Netlify, GitHub Pages, Vercel, etc.
- No server-side code required

### Python Turtle
- Requires Python environment with Turtle module
- Can be run on desktop computers
- For web deployment, would need conversion to JavaScript (e.g., using p5.js)
